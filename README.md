# cp-library-for-codon

競技プログラミング向けのcodonライブラリです。  

## 概要

[codon](https://docs.exaloop.io/)で競技プログラミングを行うための自作ライブラリです。   
API設計は[AtCoder Library(ACL)](https://github.com/atcoder/ac-library)風にしていますが、内部実装は異なります。  


## 使用方法  

必要なファイルをコピー・ペーストして使用してください。  
各ライブラリの使い方は順次追加予定です。  


## 注意事項

- 本ライブラリは、PyPy3向けに作成した自作ライブラリをcodonに移植したものです。  
移植にあたり十分なテストは行っておらず、バグが含まれる可能性があります。
- パフォーマンスが最適でない実装を含みます。
- ACLにないライブラリや、ACLとは使い方・計算量が異なるライブラリが含まれます。  
- 仕様・実装は、今後予告なく変更される可能性があります。
