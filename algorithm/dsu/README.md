# Disjoint Set Union (UnionFind)

N頂点の無向グラフに対して、
- 無向辺の追加
- 頂点の連結判定

をならし O(α(N)) で行います(α(N)は逆アッカーマン関数です)。  

## 使い方  

`DSU(N: int) -> None`
- N頂点0辺の無向グラフで初期化します。

`leader(Vi: int) -> int` 
- 頂点Viが含まれる連結成分の、現在の代表元を返します。

`merge(Ui: int, Vi: int) -> int`  
- 頂点Uiと頂点Viを結ぶ無向辺を追加し、連結成分の代表元を返します。  

`same(Ui: int, Vi: int) -> bool`
- 頂点Uiと頂点Viが連結ならTrueを、非連結ならFalseを返します。

`size(Vi: int) -> int`
- 頂点Viが含まれる連結成分のサイズ(すなわち、所属する頂点数)を返します。

`groups() -> list[list[int]]`
- 各連結成分の頂点を昇順に並べたリストを、辞書順に並べ替えた二次元リストを返します。  
- 返り値は、例として`[[0, 5], [1], [2, 3, 4]]`のようになります。  
**ACLと異なり、返り値の順序は内側のリストが番号昇順・外側のリストが辞書順で固定です。** 
- 内側のリストの先頭の頂点は、連結成分内で最も頂点番号が小さい頂点です。  
**その連結成分の代表元とは限りません。**
- 計算量: 時間・空間ともにO(N)
