# 高速素因数分解  

128bit整数の除算1回の計算量をO(L)として、期待計算量O($L × N^{\frac{1}{4}}$)で素因数分解を行います。

## 使い方

`prime.is_prime(N: int) -> bool`
- ミラー・ラビン素数判定法を用い、Nが素数か判定します。
- 制約: 1 ≤ N < $2^{63}$
- 計算量: 128bit整数の除算1回の計算量をO(L)として、O(7LlogN)

`prime.factorize(N: int) -> list[tuple[int, int]]`
- Nを素因数分解し、素因数と次数のリストを返します。   
返り値のリストの各要素は `(素因数: int, 次数: int)` の形のタプルです。
- 制約: 1 ≤ N < $2^{63}$
- 期待計算量: 128bit整数の除算1回の計算量をO(L)として、O($L × N^{\frac{1}{4}}$)

`prime.divisor(N: int) -> list[int]`
- Nの約数を昇順に列挙します。
- 制約: 1 ≤ N < $2^{63}$
- 計算量: Nの約数の個数をdとして、O(d logd) + `prime.factorize(N)`の計算量 O($L × N^{\frac{1}{4}}$)
